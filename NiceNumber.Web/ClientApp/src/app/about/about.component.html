Об игре

Первая версия игры "Закономерности" разработана и реализована в период с мая 2021 года по август 2021 года. Доработка продолжается в настоящее время.
Следующая версия игры будет включать интерактивное обучение по каждому типу закономерности.

Закономерность - последовательность цифр, группирующихся в числа, определяемая алгоритмом, по которому происходит их повторение, изменение или замещение.

Текущая версия игры включает в себя поиск следующих закономерностей:

  • Одинаковые цифры
    Пример    : для числа '171127' выделите нажатием сначала на все (три) единицы, а затем кнопку "Проверить" под в столбце под надписью "Одинаковые цифры", затем выделите нажатием на все (две) семерки и кнопку "Проверить" в том же столбце.
  • Одинаковые числа (минимум 2-значные)
    Пример    : для числа '17217' выделите нажатием первые и последние две цифры, а затем нажмите кнопку "Проверить" под в столбце под надписью "Одинаковые числа (минимум 2-значные)". Повторяющееся число - 17.
  • Зеркальные цифры
    Пример    : для числа '172741' выделите нажатием цифры 1, 7, 7, 1, а затем нажмите кнопку "Проверить" под в столбце под надписью "Зеркальные цифры". Зеркальность видна при отбрасывании всех цифр, кроме перечисленных. Зеркальные цифры не обязательно должны быть подряд идущими.
  • Кратные числа (минимум 2-значные)
    Пример    : для числа '150210' выделите нажатием цифры 5, 0, 1, 0, а затем нажмите кнопку "Проверить" под в столбце под надписью "Кратные числа (минимум 2-значные)". Кратность определяется тем, что число 50 делится нацело на 10.
  • Арифметическая прогрессия
    Пример    : для числа '2354' выделите нажатием цифры 2, 3, 4, а затем нажмите кнопку "Проверить" под в столбце под надписью "Арифметическая прогрессия". Разность прогрессии: 3-2=1, 4-3=1. 2, 3, 4 в числе расположены последовательно.
  • Геометрическая прогрессия
    Пример    : для числа '36012' выделите нажатием цифры 3, 6, 1, 2, а затем нажмите кнопку "Проверить" под в столбце под надписью "Геометрическая прогрессия". Знаменатель прогрессии: 6/3=2, 12/6=2. 3, 6, 12 в числе расположены последовательно.
Дополнительные правила
  • Закономерности считаются слева направо (кроме закономерности "Кратные числа (минимум 2-значные)")
  • Если элементы закономерности - числа, а не цифры (закономерности "Зеркальные цифры", "Одинаковые цифры"), то они должны быть неразрывными. Цифры группируются в числа слева направо.
  • Если одна закономерность одного типа включает другую закономерность такого же типа, и у них одинаковые числа-подсказки (кроме закономерностей "Одинаковые цифры" и "Одинаковые числа (минимум 2-значные)"), то считается только большая по количеству цифр закономерность.
  • Если закономерность "Одинаковые числа (минимум 2-значные)" включается все цифры закономерности "Одинаковые цифры", то считается только закономерность "Одинаковые числа (минимум 2-значные)".
  • "Кратные числа (минимум 2-значные)": если при добавлении любой цифры вперед большего из кратных чисел оно останется кратным меньшему, то считается только наименьшее из возможных больших кратных чисел.
Помощь в игре

Если вы все еще испытываете трудности в понимании того, как искать закономерности какого-либо типа:

  • в процессе игры пользуйтесь подсказками - кнопками оранжевого цвета с текстом "Подсказать" либо "?" соответствующего типа закономерности.
  • начните игру, нажмите "Завершить игру", затем подтвердите сообщение "Показать ненайденные закономерности". После того, как они отобразятся, вы сможете на отображенных в таблице игры закономерностях на примерах понять правила, по которым их надо искать.
  • По любым вопросам можете писать автору игры на один любой из контактов.
