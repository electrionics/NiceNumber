Об игре

Идея и цель игры

Каждое число по своему красиво. Увидеть это помогает умение находить закономерности. Существует много видов закономерностей, в игре представлены наиболее известные. Вам предстоит проверить и улучшить своё умение быстро их замечать. Если вы не забыли основы школьной математики - то это будет просто и интересно! А если забыли - интерактивное обучение поможет вам вспомнить! На странице рекордов вы сможете сравнить свои достижения с результатами других игроков. Потратьте время с пользой, тренируя свой интеллект, внимание, умение быстро принимать решения и мыслить стратегически!

Обучающее видео

Описание и правила

Математическая закономерность - последовательность цифр, определяемая правилом, по которому происходит их повторение, изменение или замещение.

Игра имеет три уровня сложности:

  • Легкий: число из 7-8 цифр, 6-10 закономерностей,
  • Средний: число из 9-10 цифр, 11-20 закономерностей,
  • Тяжелый: число из 11-12 цифр, 21-30 закономерностей,

и включает в себя поиск следующих закономерностей:

  • Одинаковые цифры. Пример: для числа '171127' выделите нажатием сначала все (три) единицы, а затем кнопку "Проверить" в столбце под надписью "Одинаковые цифры", затем выделите нажатием на все (две) семерки и кнопку "Проверить" в том же столбце.
  • Одинаковые числа (минимум 2-значные). Пример: для числа '17217' выделите нажатием первые и последние две цифры, а затем нажмите кнопку "Проверить" в столбце под надписью "Одинаковые числа (минимум 2-значные)". Повторяющееся число - 17.
  • Зеркальные цифры. Пример: для числа '172741' выделите нажатием цифры 1, 7, 7, 1, а затем нажмите кнопку "Проверить" в столбце под надписью "Зеркальные цифры". Зеркальность видна при отбрасывании всех цифр, кроме перечисленных. Зеркальные цифры не обязательно должны быть расположены рядом.
  • Кратные числа (минимум 2-значные). Пример: для числа '150210' выделите нажатием цифры 5, 0, 1, 0, а затем нажмите кнопку "Проверить" в столбце под надписью "Кратные числа (минимум 2-значные)". Кратность определяется тем, что при делении числа 50 на число 10 получается целое число (5).
  • Арифметическая прогрессия. Пример: для числа '2354' выделите нажатием цифры 2, 3, 4, а затем нажмите кнопку "Проверить" в столбце под надписью "Арифметическая прогрессия". Разность прогрессии: 3-2=1, 4-3=1. 2, 3, 4 в числе расположены последовательно.
  • Геометрическая прогрессия. Пример: для числа '36012' выделите нажатием цифры 3, 6, 1, 2, а затем нажмите кнопку "Проверить" в столбце под надписью "Геометрическая прогрессия". Знаменатель прогрессии: 6/3=2, 12/6=2. 3, 6, 12 в числе расположены последовательно.

Начисление очков

Очки начисляются по правилам, определяемыми типом найденной закономерности:

  • Одинаковые цифры. Количество очков = 5 х (количество цифр).
  • Одинаковые числа (минимум 2-значные). Количество очков = 10 х количество чисел.
  • Зеркальные цифры. Количество очков = 10 + 5 х (количество цифр).
  • Кратные числа (минимум 2-значные). Количество очков = 40 + 5 x (число кратности).
  • Арифметическая прогрессия. Количество очков = 10 + 5 х (разность прогрессии). Если разность прогрессии отрицательная, то в формуле вместо неё считается значение по модулю.
  • Геометрическая прогрессия. Количество очков = 20 + 10 х (знаменатель прогрессии). Если знаменатель прогрессии меньше единицы, то в формуле вместо него считается 1 / (знаменатель прогрессии).

Также в конце игры вы получаете очки, по одному за каждую оставшуюся секунду таймера, но не более половины от уже заработанных очков.

Способ начисления очков изменится в следующих версиях так, чтобы разница между "простыми" и "сложными" закономерностями не была столь существенной и максимальное число очков не так разнилось от числа к числу на одинаковом уровне сложности.

Дополнительные правила

  • Закономерности Арифметическая прогрессия и Геометрическая прогрессия строятся (но не обязательно выделяются!) слева направо. Это означает, что число-подсказка (разность/знаменатель прогрессии) будет показывать разницу между элементом справа и соседним от него слева, а не наоборот.
  • Если элементы закономерности - не цифры, как в закономерностях Зеркальные цифры, Одинаковые цифры, а числа, то они должны быть неразрывными. Цифры группируются в числа слева направо.
  • Если одна закономерность включает другую закономерность такого же типа, то считается только большая по количеству цифр закономерность. Правило действует для акономерностей Зеркальные цифры, Арифметическая прогрессия (если числа-подсказки одинаковы) и Геометрическая прогрессия (если числа-подсказки одинаковы).
  • Если закономерность Одинаковые числа (минимум 2-значные) включает все цифры закономерности Одинаковые цифры, то считается только закономерность Одинаковые числа (минимум 2-значные).
  • Кратные числа (минимум 2-значные): если при добавлении любой цифры вперед большего из кратных чисел оно останется кратным меньшему, то считается только наименьшее из возможных больших кратных чисел. Например, для числа 'XY5010', где X и Y - произвольные цифры, кратными будут считаться только 50 (наименьшее большее) и 10, т.к. при любом X и Y число Y50, как и число XY50 остается кратным 10.

Помощь в игре

Во время игры выделите цифры и нажмите кнопку "Проверить". Если закономерность не найдена - появится автоматическая подсказка с текстом "нужно удалить одну/несколько цифр" и/или "добавить одну/несколько цифр". Следуйте инструкциям подсказки, чтобы найти ближайшую закономерность выбранного типа. Если нет подсказки 'удалить' - значит все выделенные цифры являются частью ближайшей закономерности. Если нет подсказки 'добавить' - значит закономерностью являются некоторые из выделенных цифр. Количество таких подсказок неограниченно.

Число-подсказка (параметр закономерности) - число, характеризующее искомую закономерность в соответствии с её типом. Подробное описание чисел-подсказок можно просмотреть во время игры. Число-подсказка не превосходит по модулю 100, может быть равно нулю только для закономерности Зеркальные цифры, может быть меньше 1 (но не меньше 1/100) только для закономерности Геометрическая прогрессия и меньше 0 (но не меньше -100) только для закономерности Арифметическая прогрессия. Закономерности с числом-подсказкой, не входящей в указанные диапазоны - не учитывается в игре.

Если вы все еще испытываете трудности в понимании игры или некоторых закономерностей:

  • В процессе игры пользуйтесь кнопками-подсказками оранжевого цвета с текстом "Подсказать" либо "?".
  • Нажмите "Завершить игру", затем подтвердите сообщение "Показать ненайденные закономерности". Они отобразятся в таблице, и вы сможете на примерах понять правила, по которым их надо искать.
  • По любым вопросам можете писать автору игры.
